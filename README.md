### swagger api 사용법
    1. Auth 의 /auth/signup 을 통합 회원가입
    2. Auth 의 /auth/login  을 통합 로그인
    3. 로그인 후, 받은 token 을 상단 Authorize 에 입력하면 인증되어서 아래를 진행할 수 있습니다.
    4. Posts 의 /posts, post http method 를 통한 노트 생성
    5. Posts 의 /posts, get  http method 를 통한 노트 읽기
    6. /vuestagram 을 통해 이미지를 필터하여 업로드하는 기능 및
    이미지를 분류하여 어떤 이미지인지 판별 후 자동으로 태그로 등록합니다.
